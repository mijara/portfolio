Ejemplo de Tarea en Docker

El estudiante podrá correr una versión de la tarea para verificar su input/output con el del profesor, sin embargo para evitar copia de código, se dispondrá de un ejecutable en forma de imágen Docker.

Pasos para instalar Docker

  • Dirigirse https://www.docker.com/get-docker
  • Pinchar en Get Docker Community Edition
  • Pinchar en Download
  • Descargar e instalar versión correspondiente a su Sistema Operativo
  • Cuando tenga instalado y funcionando Docker, puede pasar a la siguiente parte

Iniciar Programa

Para iniciar el programa deberá tener Docker corriendo en sus sistema, abrir una terminal o línea de comandos e ingresar el siguiente comando: 

docker run -p 8000:80 mijara/gorust-gg
Luego puede ingresar a su navegador y abrir localhost:8000/cities/ para ver la lista de ciudades.
